Informations sur p5Visuel

p5Visuel est un environnement de programmation visuelle destiné au milieu scolaire (au Québec : niveaux secondaire, cegep et universitaire [formation des maîtres]). C'est un environnement général qui bénéficie d'extensions pour les mathématiques.


Environnement p5Visuel sur GitHub

En cliquant ici, vous entrez directement dans l'éditeur p5Visuel. Cet éditeur est aussi disponible sur le site profmath.uqam,ca : je l'ai dupliqué sur GitHub pour bénéficier d'un serveur sécurisé (https).

Dessinez avec votre nez sur GitHub
